# Blog App

The application is a simple blog application, wherein, the user gets the ability to :
1. Create a new blog
2.  Edit/Update the created blog
3. Delete the blog
4. See all the blogs

The application uses :
- MongoDB - Database
- Node.js / Express
- Semantic UI
- Follows RESTful routing
